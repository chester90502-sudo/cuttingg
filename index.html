Sistem SP Absensi:

  • Alpa (A) atau Ijin (I) 3x berturut-turut = SP 1
  • Setiap kelipatan 3 = SP bertambah (6x = SP 2, 9x = SP 3, dst)
  • ML (Minggu Libur) tidak dihitung dalam absensi (netral)
  • Hadir (H) = Hijau | Ijin (I) = Kuning | Alpa (A) = Merah
  • Sakit (S) = Biru | Cuti (C) = Ungu | Libur (L) = Abu-abu
